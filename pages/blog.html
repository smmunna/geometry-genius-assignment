What is DOM and what is the purpose of DOM ?

Document Object Model (DOM) is a programming interface for web documents the card's content. It represents the page so that programs can change the document structure, style, and content.


What is the purpose of DOM ?

The purpose of the DOM is to provide a platform-independent way of representing and interacting with HTML.This allows to write scripts that can work across multiple browsers

How will you select HTML elements using DOM ? Name the DOM methods.

There are several way to select HTML DOM elements. Such as:

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()
  • getElementsById()

DOM methods that can be used to select HTML elements using the DOM, such as such as setAttribute(), innerHTML, and addEventListener().

What is event bubble?

Event Bubbling ,it happens when an element receives an event, and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element.

What is a callback function and why will you use it?

A callback function is a function that is passed as an argument to another function, and is executed by that function at a later time.

We use callback function because Callbacks make sure that a function is not going to run before a task is completed but will run right after the task has completed.
